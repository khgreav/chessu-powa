Aplikace Chess je hra šachy napsaná v programovacím jazyce Java. Aplikace nabízí následující featury:
-několik tabů, každý tab spustí jednu instanci šachů
-načítání šachových partií
-ukládání šachových partií
-automatické přehrávání partií
-nastavení rychlosti automatického přehrávání
-manuální hra
-undo a redo
